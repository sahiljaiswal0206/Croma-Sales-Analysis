
# Croma Sales Analysis Dashboard

### Project Overview
This project is a Power BI dashboard built to analyze sales performance, profitability, and return trends for a retail business (Croma). The dataset is sourced from Kaggle and transformed into meaningful insights to support data-driven decision-making.

### Objective
To identify key business insights related to:

- Sales and revenue performance
- Profitability trends
- Customer behavior
- Product and regional performance
- Returns and delivery efficiency

### Tools & Technologies

- Power BI – Data visualization & dashboard creation
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – Calculated measures and KPIs

### Dashboard Features
1. Sales Overview
- Total Sales: ₹228M
- Net Revenue: ₹225M
- Net Profit: ₹55M
- Profit Margin: 24.3%
- Monthly sales trend analysis
2. Performance Analysis
- Sales by Product, Region, and Brand
- Customer segmentation (Regular, Business, Prime, First-time buyers)
- Sales contribution funnel
3. Return & Delivery Insights
- Return Rate: 23.8%
- Delivered Orders: 25.2%
- Cancelled Orders: 25.8%
- Return cost analysis by region
- Product-wise and brand-wise return trends
4. Interactive Features
- Dynamic filters (Region, Product, Quarter, Fiscal Year)
- Drill-down analysis
- User-friendly navigation across multiple report pages

### Key Insights
- Certain products like laptops and cameras contribute the highest sales.
- West and North regions generate the most revenue
- High return rates observed in specific product categories
- Customer segmentation shows strong contribution from regular and business customers
- Profit margins remain consistent despite return costs

- Applied DAX for KPI calculations and performance metrics
- Performed data cleaning and transformation using Power Query
- Designed interactive and user-friendly dashboards
